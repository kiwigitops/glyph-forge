# Glyph Forge

Glyph Forge is a self-contained browser app for sketching, generating, sharing, and exporting small mosaic tiles.

Open `index.html` in a browser. Your current tile, palette, brush, symmetry, seed, mode, and export settings are saved locally in the browser.

Features include manual painting, seeded pattern generators, compact design codes, repeating previews, and tile/sheet/wallpaper PNG exports.
